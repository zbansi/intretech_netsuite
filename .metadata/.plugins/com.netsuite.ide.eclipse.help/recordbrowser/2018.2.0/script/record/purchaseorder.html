The Approval Status field reflects the state of the transaction in the approval process. Purchase orders and requests may have one of the following statuses: * Pending Approval &#x2013; The purchase has not yet been approved. * Approved &#x2013; The purchase has been given approval for processing. * Rejected &#x2013; The purchase approval is refused and the order will not be processed. You may manually select a different status in this field if needed.
